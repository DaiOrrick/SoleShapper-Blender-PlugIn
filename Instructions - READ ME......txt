READ ME.....

{For use in Blender 5.0.1 and later.}

After download, extract .zip file into any folder of your choice.

Then ::

Open Blender → go to Edit > Preferences

Click the Add-ons tab

Click the drop-down arrow next to the Install button and choose Install from Disk...

Navigate to chosen folder and select SoleShapper.py

Tick the checkbox to enable it

Press N in the 3D viewport → look for the SoleShapper2 tab
